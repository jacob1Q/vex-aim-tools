# vex-aim-tools
Programming framework for the VEX AIM robot.

## Authors

David Touretzky, Carnegie Mellon University

vex-aim-tools is derived from cozmo-tools, written by David Touretzky, which was in turn based on the Tekkotsu framework developed by Ethan Tira-Thompson and David Touretzky.
